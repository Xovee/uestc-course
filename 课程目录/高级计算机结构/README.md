# 高级计算机结构

本课程为信息与软件工程学院所开设的研究生专业基础课，主要讲授的内容包括：
 - 现代计算机硬件体系结构, 着重介绍:
     - 处理器体系结构（主流处理器：历史到现在）
     - 采用的关键技术,  如流水线、保护模式、超标量、指令重调度、超线程、多核技术、向量计算等, 并以 Intel 处理器为典型代表
 - RISC 处理器设计方法以及与 CISC 的比较
 - 64 位处理器
 - 总线技术
 - 计算机的一般性能评价方法等
